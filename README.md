## To run
Here's how you can run this website locally:
 
- Clone this repository using the command ```git clone https://github.com/jaza-k/personal-website```
- Run ```gulp```
- Run ```NODE_ENV=production node index.js```
- Open [localhost:3000](http://localhost:3000) to view the site in your browser
